What Relay does

  • Tabbed bookmarks — group links into named tabs and open them in the browser with one click.
  • Auto-fetched favicons — icons are pulled from each site automatically and cached for 30 days; no manual setup.
  • Embedded MCP server — AI agents (Claude Code, etc.) can create, update, and remove bookmarks via a local JSON-RPC endpoint at localhost:3333.
  • .relay page viewer — create structured doc pages in XML with per-page custom CSS; navigate between them inside VS Code without leaving the editor.
  • VS Code native theming — colors, borders, and accents follow your active VS Code theme automatically. Any theme. No toggle needed.
  • Zero external dependencies — everything runs locally; no cloud sign-in, no tracking.
