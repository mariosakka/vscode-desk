User Guide

Everything you need to install Relay and start using it day-to-day.

Installation

Relay ships as a standard VS Code extension (.vsix file). Install it from the Extension Development Host during development, or from the VS Code Marketplace once published.

Install from a .vsix file

  1. 1
    Open VS Code and press Ctrl+Shift+P (macOS: ⌘⇧P) to open the Command Palette.
  2. 2
    Type Extensions: Install from VSIX… and press Enter.
  3. 3
    Browse to the relay-0.0.1.vsix file and confirm. VS Code will reload.
  4. 4
    Look for the Relay icon in the Activity Bar on the left edge of the window.

Run in development mode

  1. 1
    Clone the repository and run npm install && npm run compile.
  2. 2
    Open the folder in VS Code and press F5. A new Extension Development Host window opens with Relay active.

The Sidebar Panel

Click the Relay icon in the Activity Bar to open the sidebar. The panel has three areas:

AreaWhat it does
HeaderShows the Relay name. Colors follow your active VS Code theme automatically.
Tab barHorizontal tabs for each bookmark group. Click a tab to switch.
Bookmark gridCards for each bookmark in the selected tab. Click a card to open the URL.

Creating a Tab

Tabs are named groups that hold bookmarks. You must create at least one tab before you can add bookmarks.

  1. 1
    Open the Command Palette (Ctrl+Shift+P).
  2. 2
    Run Relay: Add Tab.
  3. 3
    Type a name for the tab (e.g. Work, Research, Tools) and press Enter.

The new tab appears immediately in the sidebar. You can create as many tabs as you like.

Adding a Bookmark

  1. 1
    Run Relay: Add Bookmark from the Command Palette.
  2. 2
    Select the tab to add it to from the quick-pick list.
  3. 3
    Enter a title for the bookmark.
  4. 4
    Enter the URL (e.g. https://example.com).
  5. 5
    Optionally enter an emoji icon. Leave blank to have Relay auto-fetch the site's favicon.
  6. 6
    Optionally enter a description. Press Enter or Escape to skip.
Tip: When you leave the icon field blank, Relay fetches the favicon from the site's /favicon.ico or <link rel="icon"> tag and stores it locally. This happens once and is cached for 30 days.

Removing Bookmarks and Tabs

Remove a bookmark

Run Relay: Remove Bookmark from the Command Palette. Pick the tab, then the bookmark. The card disappears immediately.

Remove a tab

Run Relay: Remove Tab. Removing a tab also removes all bookmarks inside it — this cannot be undone.

Note: You can also remove bookmarks directly from the sidebar by clicking the × button on a bookmark card.

Creating and Viewing Pages

Pages are .relay files stored in relay-pages/ inside the open workspace. They use a simple XML format with optional per-page CSS and render in a full VS Code editor tab.

Create a new page

  1. 1
    Run Relay: New Page from the Command Palette.
  2. 2
    Enter a title. Relay creates the .relay file in relay-pages/ and names it from the title.
  3. 3
    Edit the file in VS Code's editor, or ask an AI agent to populate it via create_page / update_page.

Open a page

  1. 1
    Run Relay: Open Page from the Command Palette and pick a page from the list.
  2. 2
    Or add a bookmark with relay-page:filename.relay as the URL — clicking the card opens the viewer directly.
Navigation: Click any .relay link inside the viewer to navigate to another page. The ← Back button traces your history. External https:// links open in your browser.

Theming

Relay reads VS Code's active color theme automatically — backgrounds, text, borders, and accent colors all adapt to whatever theme you have installed (One Dark Pro, Catppuccin, Solarized, etc.). Change your VS Code theme and Relay updates instantly; no separate toggle needed.

Settings

SettingDefaultDescription
relay.mcpPort 3333 Port the embedded MCP HTTP server listens on. Change if 3333 conflicts with another local service. Requires a VS Code window reload to take effect.

Open VS Code Settings (Ctrl+,), search for Relay, and adjust there, or add it to your settings.json:

{
  "relay.mcpPort": 4444
}
